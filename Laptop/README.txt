============================================================
  FocusFlow Lite — 智能专注度辅助系统
============================================================

【运行方式】
  双击 "FocusFlow Lite.exe" 即可启动

【首次运行提示】
  - 程序启动后会自动下载人脸识别模型 (~5MB, 仅首次)
  - 摄像头需要几秒预热，请耐心等待
  - 点击 "▶ 开始记录" 开始会话
  - 点击 "⏹ 结束并评估" 查看评估报告

【需要摄像头】
  - 笔记本内置摄像头即可
  - 外接摄像头请确认连接正常

【屏幕监控功能】
  - 需要 apikey.txt 中包含有效的 minimax API Key
  - 不提供 Key 也不影响眼动追踪功能
  - 可以从命令行运行: FocusFlow Lite.exe --camera-only
